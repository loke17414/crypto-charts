개인정보처리방침

시행일: 2026-07-22 · Orbinex

1. 수집 항목

  • 계정: 이메일, 비밀번호(해시), 약관 동의 시각, 이메일 인증 시각
  • 거래 연동: 이용자가 입력한 거래소 API 키(암호화 저장)
  • 결제: 주문번호·결제키·금액·상태 등 결제 원장, 결제 대행사(토스페이먼츠) 처리 정보
  • 이용 기록: 봇 가동·AI 사용량, 접속·오류 로그(서비스 운영에 필요한 범위)

2. 이용 목적

회원 식별, 서비스 제공, 결제·구독 관리, 보안·부정이용 방지, 고객 지원, 법령상 의무 이행.

3. 보관 및 파기

회원 탈퇴 또는 목적 달성 시 지체 없이 파기합니다. 다만 관련 법령에 따라 일정 기간 보관이 필요할 수 있습니다. 거래소 API 키는 마스터 키로 암호화하여 저장합니다.

4. 제3자 제공

법령에 따른 경우를 제외하고 이용자 동의 없이 개인정보를 제3자에게 판매하지 않습니다. 결제·이메일 발송·클라우드 인프라 등 처리 위탁이 있을 수 있으며, 위탁 시 필요한 보호조치를 합니다.

5. 이용자 권리

열람·정정·삭제·처리정지 요청이 가능합니다.

  • 계정 페이지에서 본인 데이터 JSON 내보내기
  • 같은 페이지에서 비밀번호 확인 후 계정 삭제(관련 데이터 cascade 삭제)
  • 추가 요청: support@orbinex.net

6. 안전조치

전송 구간 HTTPS, 비밀번호 해시, API 키 암호화, 접근 통제 등 합리적 보호조치를 적용합니다.

7. 문의·사업자 정보

고객센터: support@orbinex.net
